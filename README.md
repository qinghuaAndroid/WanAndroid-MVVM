# 一个基于 Kotlin + MVVM + Jetpack + DataBinding + 协程的 玩Android 客户端

## 项目架构
* Kotlin(协程) 语言
* MVVM 架构
* Jetpack 组件(ViewModel、LiveData、Navigation、Startup...)
* DataBinding
* Koin 依赖注入
* Retrofit + 协程 网络加载
* Coil 图片加载
* Realm 数据库
* Arouter 路由框架
